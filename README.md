### Who doesn't like free food right?

To run, clone the repo and run the following:

For Chick-Fil-A:
```
./script/free_cfa.sh
```

For McDonalds:
```
./script/free_mcds.sh
```

For Dairy Queen:
```
./script/free_dq.sh
```

* ruby 2.2.2p95
* Rails 5.0.6
